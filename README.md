# robot spy
Проект: Радиоуправляемая машинка-шпион с камерой

Цель проекта:
Создание полностью функциональной миниатюрной радиоуправляемой машинки с системой видеонаблюдения. Проект предназначен для демонстрации практических навыков в области схемотехники, работы с радиоканалами, управления двигателями и решения проблем электромагнитных помех.

Ключевые характеристики:
*Дальность управления: до 10 метров на открытой местности.
*Видеотрансляция: Модуль камеры на OV7670 с дальностью до 10 метров (с возможными помехами).
*Конструкция: Компактное шасси с возможностью маневрирования по "танковой" схеме.

1. Перечень основных компонентов

*Система радиоуправления :
*Пульт управления: передатчик с 4 кнопками (вперед, назад, влево, вправо).
*Приемник: модуль с 4-мя логическими выходами (по одному на каждую команду).
*Система движения:
*Драйвер моторов: MX1508 для управления двумя двигателями.
*Двигатели: 2 мотора-редуктора.
*Элементы защиты: Диоды для развязки сигнальных линий от драйвера.
*Видеосистема:
*Камера: Модуль на микросхеме OV7670 с радиопередатчиком.
*Приемники видео (варианты): портативный FPV-монитор, экран на часах, USB-приемник для смартфона.
*Система питания:
*Аккумулятор: Li-ion 18650 для силовой части (двигатели).
*Преобразователь напряжения: Повышающий модуль для питания радиоэлектроники (до 5В).
*Компоненты для подавления помех:
*Конденсаторы: Электролитические (для сглаживания низкочастотных помех) и керамические (для высокочастотных), устанавливаются параллельно цепям питания.
* Антенна: Самодельная четвертьволновая (23-30 см) для приемопередатчика 315 МГц.

2. Схема и логика управления двигателями

* Принцип "танкового хода": Каждый мотор управляется независимо для выполнения всех маневров:
*Вперед: Оба мотора вращаются вперед.
*Назад: Оба мотора вращаются назад.
*Поворот на месте: Моторы вращаются в противоположных направлениях.
*Плавный поворот: Один мотор работает, второй остановлен или работает с меньшей скоростью.
* Схема подключения:
 1.  4 логических выхода с приемника (HIGH при нажатии кнопки) подаются на входы драйвера MX1508.
 2.  Последовательно с каждым сигнальным проводом установлен защитный диод. Это предотвращает обратные токи и короткие замыкания между каналами, если на приемнике активируется несколько команд одновременно.
3.  Драйвер преобразует логические сигналы в мощные сигналы для управления моторами.

3. Конструкция шасси и механика
* (шасси): Используется готовый корпус (например, батарейный отсек от игрушки) или самодельная платформа.
*Двигатели и колеса:
*Мотор-редукторы закрепляются на шасси.
*Ведущие колеса: Устанавливаются по диагонали. Для улучшения сцепления на них надеваются резиновые кольца, вырезанные из велосипедной камеры.
*Опорные колеса (ролики): Изготавливаются из подручных материалов (например, втулок от карандашей).
*Сборка: Для прочного и быстрого крепления используется техника "суперклей + сода" (сода мгновенно полимеризует клей, создавая прочный состав).

4. Стратегия устранения помех и обеспечение стабильности
*Раздельное питание: Силовая часть (двигатели) и чувствительная электроника (радиомодуль, камера) питаются от разных источников или через разделительные фильтры. Это предотвращает просадки напряжения и помехи от работы двигателей.
*   Фильтрация цепей питания:
*   У двигателей: Керамические конденсаторы (0.1 мкФ) параллельно выводам каждого двигателя для подавления искрения щеток.
* У входов питания модулей: Электролитические (10-100 мкФ) и керамические (0.1 мкФ) конденсаторы для сглаживания пульсаций.
* Оптимизация антенны: Для приемопередатчика 315 МГц изготовлена оптимальная антенна длиной ~23 см (¼ длины волны). Для компактности она аккуратно скручена в спираль диаметром 4 мм.
*Тестирование: Работа системы проверяется как от аккумулятора, так и от лабораторного блока питания 5В для выявления проблем, связанных с просадкой напряжения.

5. Видеосистема
*   Передающая часть: Модуль камеры OV7670, оснащенный собственным маломощным видеопередатчиком (частота, например, 5.8 ГГц).
*   Приемная часть: дисплей
